Conceptos básicos de Regresión Logística

Definición

La regresión logística es un método estadístico para predecir una variable categórica binaria basándose en una o más variables predictoras. Se utiliza principalmente para problemas de clasificación binaria.

Supuestos principales

  • Variable dependiente binaria: La variable objetivo debe ser dicotómica (dos categorías)
  • Independencia de observaciones: Las observaciones deben ser independientes entre sí
  • Ausencia de multicolinealidad: Las variables predictoras no deben estar altamente correlacionadas
  • Tamaño de muestra: Se requiere un número suficiente de casos para cada categoría

Aplicaciones comunes

  • Predicción de riesgo en medicina
  • Detección de fraude
  • Predicción de asistencia a citas médicas
  • Análisis de crédito
