Conceptos básicos de Regresión Logística

Definición

La regresión logística es un modelo que se utiliza cuando la variable que queremos predecir solo tiene dos posibles resultados, como sí/no, 0/1 o verdadero/falso

Funcionamiento

Lo que hace este modelo es tomar varias variables independientes combinarlas de forma lineal y luego aplicar una función llamada logit que transforma esa combinación en un valor real

Después se usa la función sigmoide para convertir ese valor en una probabilidad entre 0 y 1 lo que nos permite interpretar el resultado como la probabilidad de que ocurra uno de los dos posibles eventos.

Según la documentación de Scikit-learn (2023) esta transformación permite modelar la probabilidad de pertenencia a una clase binaria de forma precisa y GeeksforGeeks (2023) lo presenta como una herramienta útil para problemas como detección de fraude o predicción médica.

Brixius (2016) explica que el logit convierte probabilidades en valores reales y la sigmoide hace el trabajo inverso es decir devolviéndolos al rango [0, 1] Además Chawla (2023) dice que el uso de la sigmoide en este modelo tiene una base matemática sólida derivada de la probabilidad condicional en distribuciones Bernoulli.
