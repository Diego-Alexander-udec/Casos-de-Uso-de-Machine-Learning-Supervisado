Conceptos básicos de Regresión Lineal

Definición

La regresión lineal es un método estadístico utilizado para modelar la relación entre una variable dependiente y una o más variables independientes, asumiendo que esta relación es lineal.

Supuestos habituales

  • Linealidad: En el conjunto de datos a analizar debe existir una relación lineal, es decir entre la variable independiente X y la variable dependiente Y, los analistas crean un gráfico de dispersión para verificar si los valores caen a lo largo de una línea recta, de lo contrario se pueden aplicar funciones no lineales para crear matemáticamente la relación lineal entre las variables
  • Independencia: La independencia residual es la que utilizan los analistas para medir la precisión de la predicción mediante un residuo, el residuo es la diferencia entre los datos observados y el valor esperado, los residuos no deben seguir algún patrón en específico, se puede utilizar diferentes pruebas matemáticas como la de Durbin-Watson para determinar la independencia residual
  • Homocedasticidad: Se dice que los residuos tienen una variación constante o desviación estándar de la media para cada valor de X, de lo contrario es posible que el resultado del análisis no sea preciso, sino se cumple esta suposición es posible que se tenga que cambiar la variable dependiente ya que la variación se produce de manera natural.
  • Normalidad del error:Al momento de graficar esto nos puede ayudar a verificar si los residuos se distribuyen normalmente, los residuos deben caer a lo largo de una línea diagonal en el centro de la gráfica, sino están normalizados se prueban los datos para detectar valores atípicos aleatorios o valores que no sean típicos.
