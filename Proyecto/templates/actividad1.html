Investigación Casos de Uso Machine Learning Supervisado
Caso 1: Diagnóstico Médico Asistido

Industria: Salud

Empresa: IBM Watson Health

Algoritmo utilizado: Redes Neuronales Convolucionales (CNN), especializadas en clasificación de imágenes médicas.

Problema resuelto: La dificultad para detectar enfermedades como el cáncer en etapas tempranas, debido a la complejidad de las imágenes médicas y la variabilidad en la interpretación humana.

Beneficios: Mayor precisión diagnóstica, Reducción de errores humanos y Intervenciones más oportunas

Ejemplo encontrado: Acelerar el diagnóstico de tipos complejos de cáncer link del articulo: https://techgolly.com/ai-in-healthcare-ibm-watsons-contribution-to-cancer-research

Referencia APA: IBM. (2021). Watson Health: Transforming healthcare with AI. https://www.ibm.com/watson-health

Caso 2: Automatización de Clasificación de Correos

Industria: Tecnología / Automatización

Empresa: Levity

Algoritmo utilizado: Algoritmos de Procesamiento de Lenguaje Natural y Clasificación de Texto

Problema resuelto: La categorización manual de correos electrónicos y tiempo perdido en respuestas repetitivas.

Beneficios:

  • Categorización automática de correos
  • Respuestas personalizadas automatizadas
  • Mejora en tasas de conversión
  • Reducción significativa del tiempo de procesamiento

Ejemplo encontrado: Sistema de clasificación automática de correos que categoriza y genera respuestas personalizadas basadas en el contenido.

Referencia: Levity. (2023). Email Automation with Machine Learning. https://levity.ai/use-cases/email-automation

Referencia adicional: IBM. (2023). Casos de uso de machine learning. https://www.ibm.com/mx-es/think/topics/machine-learning-use-cases

Caso 3: Predicción de Viajes Compartidos

Industria: Tecnología / Transporte y Navegación

Empresa: Waze (utilizando la tecnología de Google Cloud)

Algoritmo utilizado: Gradient Boosted Regression Trees y Google Cloud AI ademas de logistic regression

Problema resuelto: Se presenta la baja probabilidad de éxito al conectar pasajeros con conductores para viajes compartidos (carpooling)

Beneficios:

  • Aumento en la tasa de éxito de los viajes compartidos
  • Mejora la experiencia del usuario
  • Optimización del proceso de emparejamiento
  • Sistema de predicciones rápido y escalable

Ejemplo encontrado: Sistema que analiza en tiempo real aproximadamente 90 factores para predecir la probabilidad de que un viaje se concrete, ordenando la lista de conductores disponibles mostrada a los pasajeros.

Referencia APA: Regev, B. (2019, 10 de octubre). How Waze predicts carpools using Google Cloud AI Platform. Google Cloud Blog. https://cloud.google.com/blog/products/ai-machine-learning/how-waze-predicts-carpools-using-google-cloud-ai-platform

Caso 4: Piloto Automático Tesla FSD

Industria: Automotriz / Inteligencia Artificial

Empresa: Tesla

Algoritmo utilizado: Aprendizaje Supervisado con Redes Neuronales Convolucionales para procesamiento de imágenes

Problema resuelto: La complejidad en la toma de decisiones autónomas en la conducción debido a la variabilidad en infraestructura vial y situaciones de tráfico impredecibles.

Beneficios:

  • Mejora continua del sistema mediante aprendizaje de datos reales
  • Capacidad de adaptación a diferentes entornos de conducción
  • Procesamiento en tiempo real de información visual
  • Sistema escalable con retroalimentación constante

Ejemplo encontrado: Sistema de conducción autónoma que utiliza cámaras y procesamiento de imágenes para identificar objetos, señales de tránsito y tomar decisiones de conducción en tiempo real.

Referencia APA: Singh, K. (2025). Inside Tesla's FSD Patent Explains How FSD Works. NotaTeslaApp. https://www.notateslaapp.com/news/2362/inside-teslas-fsd-patent-explains-how-fsd-works
